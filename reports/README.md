# Reports

PDF report is generated by running:

```bash
python scripts/generate_report.py --runs-dir runs --out-path reports/project_report.pdf
```

This requires the following runs to have been completed first:
- `runs/data_profile/` (plot_dataset.py)
- `runs/resnet18/` (train_finetune.py)
- `runs/resnet18/eurosat_eval/` (evaluate.py)
- `runs/resnet18/uc_merced_eval/` (evaluate.py)
- `runs/change_detection/` (run_change_detection.py)
- `runs/change_detection/heatmaps/` (save_change_heatmaps.py)
- `runs/spatial_leakage/` (spatial_leakage_experiment.py)
- `runs/resnet18/error_analysis/` (visualize_errors.py)
